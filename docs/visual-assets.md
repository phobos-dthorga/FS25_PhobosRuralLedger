# Visual Assets

## Current State

`mod/icon.dds` is a generated placeholder so the mod package is structurally
complete from the first commit. Replace it before any public gameplay release.

## Desired Icon Direction

The final icon should communicate:

- rural economy;
- land records or a ledger;
- grounded FS25 realism;
- subtle Phobos identity.

Possible motifs:

- a field map overlaid with a ledger page;
- a brass account book clasp and furrow lines;
- land parcels with debit/credit marks;
- a farm auction notice pinned over a balance sheet.

## Asset Policy

- Keep source art in an `assets/source/` folder once real art exists.
- Store game-ready DDS assets in `mod/`.
- Do not copy third-party art into the repo unless the license explicitly allows
  it and attribution is recorded.
